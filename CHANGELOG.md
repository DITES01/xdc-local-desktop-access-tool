## v1.0.0 — Initial Public Release

### Features
- Balance checking via XDC RPC
- Local transaction signing
- Keystore JSON generation
- BIP39 seed phrase wallet creation and recovery
- Address derivation and scanning
- Local authorisation workflow
- Configurable gas settings (Auto / Custom)
- Direct explorer access via XDCScan

### Security
- Local-first signing model
- Private keys never leave the user's machine
- SHA256 checksum verification
- PGP signed release files
- Public signing key provided for verification

### Notes
Initial release of the XDC Local Desktop Access Tool.

Users should always verify release files before running wallet software.

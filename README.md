# XDC Local Desktop Access Tool

A local-first desktop utility for interacting with the XDC network.

The tool is designed to provide secure local signing and wallet functionality without relying on web-based wallets.

## Features

- Balance checking
- Local transaction signing
- Keystore generation
- BIP39 seed phrase support
- XDC network interaction

## Security Philosophy

This tool is designed to run fully locally. Private keys never leave the machine.
